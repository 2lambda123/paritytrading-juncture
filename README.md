# Juncture

Juncture is a fast trading protocol library for the JVM.

Juncture requires Java Runtime Environment (JRE) 8 or newer.

## Features

Juncture implements connectivity to the following trading venues:

- [**Nasdaq**](libraries/nasdaq)
